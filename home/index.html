# 多平台插件

插件名称 链接 平台 简介
✅ (opens new window)GriefDefender (opens new window) (opens new window) Sponge 1.12.2 领地插件，全UI操作、2D/3D圈地模式、领地方块、情境定义、行为包装等
Chunky (opens new window) (opens new window) (opens new window) Sponge Fabric Forge 世界预生成插件，简约高效，功能丰富，支持多个不同的平台。
LibertyBans (opens new window) (opens new window) Bungge Velocity 封禁插件，简约设计，性能优异，轻量高效。
ServerUtils (opens new window) (opens new window) Bungee Velocity 插件管理，Plugman 理念的继承与延展。
UltimateChat (opens new window) (opens new window) Bukkit Sponge Velocity 聊天插件，支持平台丰富，Json 支持，标签化设计友善。
ajQueue (opens new window) (opens new window) Bungge Velocity 群组排队插件，队列优先级，队列之间独立运行，同一玩家参与多个队列。
Prism (opens new window) (opens new window) (opens new window) Sponge 记录回滚插件，功能丰富，性能优异，用户友好。
